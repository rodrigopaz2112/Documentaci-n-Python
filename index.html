Introducción

Python es un lenguaje de programación versátil y fácil de aprender.

Se utiliza en desarrollo web, análisis de datos, inteligencia artificial, entre otros.

Instalación

Python se puede descargar desde su sitio web oficial.

También puedes usar gestores de paquetes como apt o brew.

sudo apt install python3
Sintaxis Básica

La sintaxis de Python es clara y concisa.

Los bloques se definen por indentación.

print("Hola mundo") if True: print("Verdadero")
  • Indentación obligatoria
  • No se usan llaves
  • Sensible a espacios
Tipos de Datos

Python tiene varios tipos de datos incorporados.

Los más comunes son: int, float, str, list, dict.

mi_lista = [1, 2, 3] mi_dict = {"clave": "valor"}
  • int: enteros
  • float: decimales
  • str: texto
  • list: listas
  • dict: diccionarios
Control de Flujo

Permite decidir qué código se ejecuta y cuántas veces.

Incluye condicionales y bucles.

if x > 10: print("Mayor a 10") for i in range(5): print(i)
  • if/else
  • for
  • while
  • break
  • continue
Funciones

Las funciones permiten reutilizar código.

Se definen con la palabra clave def.

Pueden tener parámetros y devolver valores.

Mejoran la legibilidad y mantenimiento.

También existen funciones lambda.

  • def
  • return
  • lambda
  • args/kwargs
  • funciones anónimas
def suma(a, b): return a + b
Manejo de Errores

Python maneja errores con try y except.

Permite controlar excepciones sin que el programa falle.

También existen else y finally.

Se pueden lanzar errores con raise.

El manejo correcto evita errores inesperados.

  • try
  • except
  • else
  • finally
  • raise
try: x = 1 / 0 except ZeroDivisionError: print("Error")
Módulos y Paquetes

Permiten organizar código en archivos reutilizables.

Los paquetes agrupan varios módulos.

Se importan con import o from ... import ...

El archivo __init__.py define un paquete.

Facilita el mantenimiento del proyecto.

  • import
  • from ... import
  • __init__.py
  • pip
  • paquetes externos
import math print(math.pi)
